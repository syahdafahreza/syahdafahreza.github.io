Tentang.

Sebuah website sederhana yang berisikan info, tips, database software, game dan sebagainya. Perlu diingat bahwa website ini masih dalam tahap pengembangan. Jadi, jika masih terdapat banyak kekurangan konten/apapun mohon untuk dimaklumi. Saya akan berusaha sebisa & semampu saya untuk menambah konten lebih banyak lagi agar website ini semakin asyik & nyaman untuk dikunjungi 🙂

Alasan dan Tujuanku membuat website ini

Untuk berbagi pengalaman yang pernah ku alami serta hal yang lain seperti: Game, Aplikasi / Software, Music, Cerita, Gambar dan lain sebagainya, Aku membuat website ini dengan tujuan untuk menghibur pembaca sekalian di saat gabut, Agar lebih tahu seputar dunia Komputer, dan untuk tempat arsip file file apapun.
